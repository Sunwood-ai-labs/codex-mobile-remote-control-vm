# v0.1.0 リリースノート

![Codex Mobile Remote Control VM v0.1.0 release header](/releases/release-header-v0.1.0.svg)

v0.1.0 は **Codex Mobile Remote Control VM** の初回公開リリースです。SSHだけのUbuntu VMを、ChatGPT/Codexモバイルアプリから制御できるCodex Desktopホストへ整えるための一連の手順をSkillとしてまとめました。

## ハイライト

- mobile controlを最重要の成功条件として扱い、スマホから制御している証跡画像を同梱しました。
- `SKILL.md` は、Proxmox/SSH棚卸し、Ubuntu desktop修復、Codex Desktop起動、feature flag設定、mobile pairing、handoffまで案内します。
- `references/runbook.md` は、XFCE/LightDM、`qemu-guest-agent`、launcher作成、app-server feature state、enrollment確認、Linuxポート特有の挙動をコマンド単位で整理しています。
- CIでSkill package、helper scripts、README links、VitePress docs buildを検証します。
- GitHub Pagesで、install、usage、troubleshooting、setup proof、このrelease noteを日英で公開します。

## 同梱されたもの

- `SKILL.md`: CodexがこのSkillを使う条件とworkflow。
- `references/runbook.md`: Proxmox/SSHベースのVMセットアップrunbook。
- `scripts/audit-codex-remote-vm.sh`: 既存VMを変更せずに状態確認する監査スクリプト。
- `README.md` / `README.ja.md`: install、use、validation、proofを含む公開README。
- `docs/`: 英日VitePress documentation。
- `assets/screenshots/` / `docs/public/screenshots/`: mobile controlとProxmoxの証跡画像。
